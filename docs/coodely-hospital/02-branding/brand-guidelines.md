## Brand Identity

### Logo
- **Primary Logo**: Coodely Hospital (to be designed)
- **Icon Mark**: Medical cross + heartbeat line (suggested)
- **Typography**: Modern, clean sans-serif

### Brand Values
1. **Trust** - Reliable healthcare management
2. **Care** - Patient-centered approach
3. **Innovation** - Modern technology for better healthcare
4. **Accessibility** - Easy to use for everyone
5. **Security** - HIPAA/GDPR compliant, data protection

### Tone of Voice
- **Professional** but approachable
- **Clear** and jargon-free
- **Empathetic** and supportive
- **Confident** without being arrogant

## Typography

### Primary Font: Inter
- **Headings**: Inter Bold (700)
- **Subheadings**: Inter Semibold (600)
- **Body**: Inter Regular (400)
- **Captions**: Inter Medium (500)

### Fallback Stack
```css
font-family: 'Inter', 'Segoe UI', 'Roboto', 'Helvetica Neue', sans-serif;
```

### Font Sizes
- H1: 2.5rem (40px)
- H2: 2rem (32px)
- H3: 1.5rem (24px)
- H4: 1.25rem (20px)
- Body: 1rem (16px)
- Small: 0.875rem (14px)

## Iconography

### Icon Style
- **Style**: Outline/Line icons
- **Weight**: 2px stroke
- **Corners**: Rounded

### Medical Icons

- Pills (Pharmacy/Prescriptions)
- Clipboard (Medical Records)
- Calendar (Appointments)
- Test Tube (Lab)
